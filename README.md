# 🌱 AgroForte | Sustentabilidade no Campo

O **AgroForte** é uma página web institucional e interativa dedicada a promover práticas agrícolas sustentáveis. O objetivo do projeto é conscientizar produtores e entusiastas sobre a importância de equilibrar a produção de alimentos com a preservação ambiental, incentivando o uso de menos componentes químicos e mais soluções naturais.

---

## 🚀 Funcionalidades

* **Design Responsivo:** Interface totalmente adaptada para visualização em computadores, tablets e smartphones.
* **Seção Informativa:** Apresentação de práticas ecológicas essenciais, como rotação de culturas, compostagem e captação de água da chuva.
* **Calculadora Interativa:** Uma ferramenta em JavaScript que calcula a quantidade estimada (em kg) de adubo orgânico necessário com base no tamanho da área informada pelo usuário.
* **Navegação Suave (Smooth Scroll):** Experiência de usuário aprimorada com rolagem suave entre as seções da página.

---

## 🛠️ Tecnologias Utilizadas

O projeto foi desenvolvido utilizando as tecnologias web fundamentais:

* **HTML5:** Estruturação semântica do conteúdo.
* **CSS3:** Estilização moderna, uso de variáveis nativas, Flexbox para alinhamento e Media Queries para responsividade.
* **JavaScript (ES6):** Lógica da calculadora, validação de dados de entrada e manipulação do DOM.
* **Google Fonts:** Utilização da fonte *Poppins* para uma tipografia limpa e profissional.
